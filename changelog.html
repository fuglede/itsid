Changelog

What's New in Itsid

Every change we have ever made to our product. The output remains identical to the input.

v3.1

March 31, 2026 Latest
  • + Added Chat interface with conversation history, LocalStorage persistence, and model selector.
  • + Advanced Identity Mode™ now available on Pro tier. Behaviour is identical to standard Identity Mode.
  • + Improved streaming throughput. Output now emerges at up to 1.5 tok/s on standard tier, up from 1.5 tok/s.
  • ~ Fixed: Edge case where output was not identical to input in 0.000001% of cases. Root cause: Unicode normalization of U+FEFF (zero-width no-break space) during tokenization. Resolved by removing the identity function entirely and reimplementing it from scratch.
  • ~ Fixed: A fidelity regression introduced in v3.0.2 where very long inputs consisting entirely of whitespace were returned with trailing newline stripped. This was a bug. Whitespace is data.

v3.0

March 2025
  • + Upgraded from 7B to 14B parameters. Output is identical to v2.x output.
  • + Context window expanded to unlimited. Previously capped at 2M tokens, which we now consider embarrassingly small for an identity function.
  • + Provenance certificates now include RFC 3161 timestamps and a SHA-256 hash of both input and output. The hashes are identical.
  • − Removed experimental "Lossy Identity Mode" beta. It was a mistake. We do not discuss it further.

v2.4

September 2024 Deprecated
  • + REST API now accepts application/octet-stream in addition to JSON. Binary input is returned as binary output.
  • ~ Fixed: In rare cases involving inputs of exactly 4,096 bytes, the model would append a single null byte (0x00) to the output. Fixed. The null byte was ours. We apologize.

v1.0

June 2024
  • + Initial public release. The model returns its input. This is the entirety of the feature set. We consider it complete.
